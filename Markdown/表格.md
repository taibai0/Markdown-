# Markdowm 表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。  
语法格式如下:  

```
| 表头1 | 表头2 | 表头3 |
| --- | --- | --- |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |
```

效果如下:

| 表头1 | 表头2 | 表头3 |
| ----- | -----| -----|
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |

注：表头可以省略，但建议保留。

##### 对齐方式

表格单元格默认左对齐，如果需要右对齐或居中对齐，可以使用 colons 来设置对齐方式。

```
| 左对齐 | 右对齐 | 居中对齐 |
| :-- | :--: | :--: |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |
```

效果如下:

| 左对齐 | 右对齐 | 居中对齐 |
| :-- | --: | :--: |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |
