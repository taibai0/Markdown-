# Markdown 列表

Markdown 支持有序列表和无序列表。   
无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容：
* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项

有序列表则使用数字接着一个英文句点作为列表标记，后面要添加一个空格，然后再填写内容：
1. 第一项
2. 第二项
3. 第三项

### 列表嵌套

列表可以嵌套，一个列表项目可以包含一个或多个子列表项目，子列表项目用缩进表示。

例如：
1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第二个元素

### 列表的语法

Markdown 列表语法简单，基本格式如下：

```
- 内容
+ 内容
* 内容
```

其中，`-`、`+`、`*` 符号后面要加一个空格。

如果要在列表项目中添加段落，则在段落前后分别加上至少三个空格。
```
- 第一项

    第一项嵌套的第一个段落。

    第一项嵌套的第二个段落。

- 第二项

    第二项嵌套的第一个段落。

    第二项嵌套的第二个段落。
```
效果如下：

- 第一项

    第一项嵌套的第一个段落。

    第一项嵌套的第二个段落。

- 第二项

    第二项嵌套的第一个段落。

    第二项嵌套的第二个段落。
